# Security

## Reporting a vulnerability

Please report security-sensitive issues privately through GitHub's security advisory feature for this repository. Do not include private datasets, credentials, patient information, or unpublished research data in a public issue.

## Distribution trust

SpatialScope releases are distributed outside the Mac App Store without Apple notarization. macOS therefore requires a one-time user approval before the first launch. Update archives are independently authenticated with Sparkle EdDSA signatures, and SHA-256 checksums are attached to every GitHub release.

Only download SpatialScope from this repository's GitHub Releases page.
