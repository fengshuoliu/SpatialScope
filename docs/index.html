Version 1.0 · macOS 13+ · Apple Silicon and Intel

Previously TME Spatial

SpatialScope is the renamed, native successor to the published Streamlit prototype. The legacy repository and website remain available for reproducibility and existing article links.

Nine connected stages

One analysis workspace

Each stage preserves parameters and writes figures, masks, tables, and metadata into a predictable output structure.

  1. 01Inputs & CalibrationFolders, channels, and spatial scale
  2. 02Composite PreviewMultiplex and split channels
  3. 03Nuclei SegmentationDetection and parameter screening
  4. 04Cell Type AssignmentRule-based marker phenotypes
  5. 05Neighborhood AnalysisLocal cell composition
  6. 06Region AnalysisComputational and manual ROIs
  7. 07Cell DistributionBoundary bands and density
  8. 08Distance AnalysisCell and boundary distances
  9. 09Results & ExportsReview the generated package

Install from GitHub

1

Download the current DMG and drag SpatialScope into Applications.

2

On first launch, approve SpatialScope under System Settings > Privacy & Security.

3

Use Check for Updates from the SpatialScope menu for later EdDSA-verified releases.
